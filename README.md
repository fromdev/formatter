# Formatter

A simple project created in Node.js to format content in HTML. It uses some pre-defined formatting to speed up HTML creation.

## Running Locally

Make sure you have [Node.js](http://nodejs.org/) and the [Heroku CLI](https://cli.heroku.com/) installed.

```sh
$ git clone git@github.com:fromdev/formatter.git # or clone your own fork
$ cd formatter
$ npm install
$ npm start
```

Your app should now be running on [localhost:5000](http://localhost:5000/).

## Deploying to Heroku

```
$ heroku create
$ git push heroku master
$ heroku open
```
